# Танчики Мультиплеер - Деплой на Amvera

## Быстрый старт

### 1. Подготовка проекта
Проект уже готов к деплою на Amvera.

### 2. Деплой на Amvera

#### Через Git (рекомендуется)
1. Создай репозиторий на GitHub
2. Инициализируй git в папке `tanks-amvera`:
```bash
cd tanks-amvera
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <твой-репозиторий>
git push -u origin main
```

3. Зайди на [amvera.ru](https://amvera.ru)
4. Создай новый проект
5. Подключи свой GitHub репозиторий
6. Amvera автоматически обнаружит `amvera.yml` и задеплоит проект

#### Через CLI
```bash
cd tanks-amvera
amvera deploy
```

### 3. После деплоя
- Amvera выдаст тебе публичный URL типа `https://твой-проект.amvera.app`
- Отправь эту ссылку друзьям
- Все могут играть без дополнительных настроек!

## Технические детали

### Требования
- Node.js 18+
- npm

### Зависимости
- express - веб-сервер
- socket.io - реалтайм коммуникация
- three.js - 3D графика (CDN)
- simple-peer - WebRTC голосовой чат (CDN)

### Порты
- Amvera автоматически назначает порт через `process.env.PORT`
- По умолчанию 3000 для локальной разработки

## Troubleshooting

### Голосовой чат не работает
- Проверь что браузер поддерживает WebRTC
- Разреши доступ к микрофону
- Если не работает через NAT - это ограничение WebRTC P2P

### Игроки отключаются
- Проверь стабильность интернета
- Amvera может перезапускать приложение при обновлениях

### Лаги в игре
- Проверь пинг до сервера
- Amvera бесплатный план может иметь ограничения

---

## Лимиты

- Максимум 20 игроков одновременно
- Сообщения в чате до 100 символов
- История чата - последние 50 сообщений
